# 草稿与参考资料规则

## 文件名

草稿要明确标注草稿状态，不要命名得像已确认事实。

推荐：

```text
{主题}草案.md
{主题}探索.md
{主题}参考资料.md
{主题}中文翻译.md
```

## 头部模板

```markdown
# {主题}

**记录时间**：YYYY-MM-DD HH:mm:ss UTC+08:00  
**记录人**：{智能体名称或用户名称}  
**当前状态**：草稿 / 探索中 / 待审核 / 原始材料  
**内容概要**：{一句话说明这份草稿是什么}  
**信息来源**：{用户输入 / 外部资料 / 旧文档迁移 / 本轮对话整理}  

## 更新记录

| 时间 | 更新人 | 更新明细 |
| --- | --- | --- |
| YYYY-MM-DD HH:mm:ss UTC+08:00 | {智能体或用户} | 初始记录。 |
```
