# 已确认上下文文档规则

## 文件名

`context/` 下的稳定入口不默认加日期，文件名直接写主题。

推荐：

```text
产品需求文档.md
技术字段查阅字典.md
图标使用指南.md
```

## 头部模板

```markdown
# {主题}

**更新时间**：YYYY-MM-DD HH:mm:ss UTC+08:00  
**更新人**：{智能体名称或用户名称}  
**当前状态**：已确认  
**内容概要**：{1-3 句话说明本文作为长期上下文入口覆盖什么}  
**信息来源**：{用户确认 / 源项目验证 / 已完成开发结果 / 已确认旧文档迁移}  

## 更新记录

| 时间 | 更新人 | 更新明细 |
| --- | --- | --- |
```
